# devnews

## Build Setup

```bash
# install dependencies
$ npm install


# Set GOOGLE_API_KEY, GOOGLE_CLIENT_ID environmental variable to 
# Google Firebase Auth Key generated for your Firebase app and OAuth Client ID generated from GCP Console
$ set GOOGLE_API_KEY=<FIREBASE AUTH KEY> # export GOOGLE_API_KEY=<FIREBASE AUTH KEY> - for linux based OS.
$ set GOOGLE_CLIENT_ID=<Google OAuth 2 CLIENT ID> # export GOOGLE_CLIENT_ID=<Google OAuth 2 CLIENT ID>  - for linux based OS.

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
